Jour Template

Modern Login & Dashboard HTML Template

1. Introduction & Overview

Thank you for purchasing Jour! This is a clean, responsive HTML5 template focused heavily on user authentication and dashboard UI. What sets Jour apart is the inclusion of functional Firebase JavaScript logic to help you get your authentication system up and running instantly.

3. Firebase Setup Guide

To make the login, register, and password reset forms work, you must connect the template to your own free Firebase account.

Note: You must complete these steps before the authentication forms will function correctly.

Step 1: Create a Firebase Project

  1. Go to the Firebase Console.
  2. Click "Add Project" and follow the on-screen instructions.
  3. Once your project is ready, click on the Web icon () to register a web app.

Step 2: Enable Authentication

  1. In your Firebase dashboard sidebar, click on Authentication.
  2. Go to the Sign-in method tab.
  3. Enable Email/Password and click Save.

Step 3: Add Your Credentials to the Code

  1. Go to your Firebase Project Settings (the gear icon top left).
  2. Scroll down to the "SDK setup and configuration" section and copy your firebaseConfig object.
  3. Open assets/js/firebase.js in any code editor.
  4. Replace the dummy config with your actual credentials:
// Your web app's Firebase configuration
const firebaseConfig = {
    apiKey: "YOUR_API_KEY",
    authDomain: "YOUR_PROJECT.firebaseapp.com",
    projectId: "YOUR_PROJECT_ID",
    storageBucket: "YOUR_PROJECT.appspot.com",
    messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
    appId: "YOUR_APP_ID"
};

// Initialize Firebase
const app = initializeApp(firebaseConfig);
const auth = getAuth(app);

4. Customization Guide

Jour is designed to be easily modified. All styling is centralized in a single CSS file.

  • Colors: Open assets/css/styles.css. Look for the :root variables at the very top of the file to change the primary brand colors globally.
  • Images: The images included in the download are solid color placeholders dynamically generated via DummyImage to comply with marketplace rules. You can replace the <img src="..."> tags in the HTML files with your own JPG/PNG image paths to customize the look.
