# 🔐 Jour Authentication Template

Version 1.0.0

A modern Firebase-powered authentication template featuring user registration, login, password recovery, and a responsive user dashboard. Built with HTML5, CSS3, JavaScript, Firebase Authentication, and Firestore.

---

## Features

* User Registration
* Secure Login
* Password Recovery
* Password Reset
* User Dashboard
* Profile Management
* Avatar Upload
* Password Strength Validation
* Session Management
* Responsive Design
* Firebase Authentication Integration
* Firestore Database Integration
* Modern UI Design
* Cross-Browser Compatibility

## Quick Start

### Step 1

Create a Firebase project from the Firebase Console.

### Step 2

Enable:

* Authentication → Email/Password
* Firestore Database

### Step 3

Open:

```text
assets/js/firebase.js
```

Replace the placeholder configuration values with your Firebase credentials.

### Step 4

Update the password reset URL in:

```text
assets/js/forgot-password.js
```

### Step 5

Upload the files to your hosting provider or deploy using Firebase Hosting.
