3. Firebase Setup Guide

To make the authentication forms work, you must connect the template to your own free Firebase account.

Important: Complete these steps before the forms will function correctly.

Step 1: Create Firebase Project

  1. Go to Firebase Console
  2. Click "Add Project" and follow on-screen instructions
  3. Once ready, click the Web icon (</>) to register a web app

Step 2: Enable Email/Password Authentication

  1. In Firebase dashboard, click Authentication
  2. Go to Sign-in method tab
  3. Enable Email/Password and click Save
  4. Create Firestore Database (in test mode for development)

Step 3: Get Your Firebase Config

  1. Go to Firebase Project Settings (gear icon, top left)
  2. Scroll to "SDK setup and configuration"
  3. Copy your firebaseConfig object

Step 4: Add Your Credentials

  1. Open assets/js/firebase.js in your code editor
  2. Replace the dummy config with your actual Firebase credentials:
const firebaseConfig = {
    apiKey: "YOUR_API_KEY",
    authDomain: "YOUR_PROJECT.firebaseapp.com",
    projectId: "YOUR_PROJECT_ID",
    storageBucket: "YOUR_PROJECT.appspot.com",
    messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
    appId: "YOUR_APP_ID"
};

Step 5: Update Hosting URL

Open assets/js/forgot-password.js and replace:

const actionCodeSettings = {
    url: "DROP YOUR HOSTING URL HERE/reset-password.html",
    handleCodeInApp: false,
};

Example: https://your-project.web.app/reset-password.html

Step 6: Deploy to Firebase

npm install -g firebase-tools
firebase login
firebase deploy --only hosting

5. Customization Guide

Jour is designed to be easily customized. All styling is centralized in a single CSS file.

Change Colors

Open assets/css/styles.css and modify the :root variables at the top:

:root {
    --brand-purple: #6c63ff;      /* Change this */
    --brand-black: #000000;
    --text-main: #444444;
    --bg-body: #f4f7f6;
    --bg-surface: #ffffff;
    --border-light: #eaeaea;
}

Change Fonts

The template uses the "Poppins" font from Google Fonts. To change:

  1. Open index.html (and other HTML files)
  2. Find the Google Fonts link in <head>
  3. Replace with your preferred font from Google Fonts
  4. Update font-family in CSS to match

Replace Images

Add your own images to assets/img/ folder and update image paths in HTML files:

<img src="assets/img/your-image.jpg" alt="Description">

Add Custom Fields

To add more fields to registration, edit assets/js/register.js:

await setDoc(doc(db, "users", user.uid), {
    firstName,
    lastName,
    phone: document.getElementById("phone").value,  // Add custom field
    company: document.getElementById("company").value,
    // ... other fields
});

6. FAQ

Can I use this with my own domain?

Yes! You can use a custom domain with Firebase Hosting. Go to Firebase Console → Hosting → Connect domain.

Is this secure?

Yes! It uses enterprise-grade Firebase Authentication with encrypted passwords, secure session tokens, and SSL/TLS encryption.

Can I add social login?

Yes! Firebase supports Google, Facebook, GitHub, Apple, and Twitter login. Enable in Firebase Console → Authentication → Sign-in method.

What are the Firebase costs?

Firebase has a generous free tier. You get free email auth, 1GB Firestore storage, and 10GB/month hosting. Costs only apply if you exceed free limits.

Can I export user data?

Yes! Firebase provides data export tools in the Console. You can export Firestore data as JSON or CSV.

Can I use this for a SaaS?

Yes! This template is designed for SaaS applications. It's production-ready and scalable with Firebase.
